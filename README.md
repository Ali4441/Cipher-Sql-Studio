# Cipher SQL Studio

Cipher SQL Studio is a **web-based SQL sandbox** that allows users to write and execute SQL queries in a clean editor and instantly view the results.
It is designed for students and beginners to **practice SQL queries interactively**.

---

## Features

* SQL Editor powered by **Monaco Editor**
* Execute SQL queries in real-time
* Display query results in a structured results panel
* Simple and clean user interface
* Practice SQL queries without setting up a complex environment

---

## Tech Stack

**Frontend**

* React
* Vite
* Monaco Editor
* SCSS

**Backend**

* Node.js
* Express.js

## Installation & Setup

### 1. Clone the repository


git clone https://github.com/Ali4441/Cipher-Sql-Studio.git


### 2. Navigate to project folder


cd Cipher-Sql-Studio




## Backend Setup

Open terminal and run:


cd server
npm install
nodemon server.js


Backend server will start at:


http://localhost:5000




## Frontend Setup

Open a new terminal and run:


cd client
npm install
npm run dev
```

Frontend will start at:

```
http://localhost:5173



## How to Use

1. Start the backend server.
2. Start the frontend application.
3. Open the frontend in your browser.
4. Write a SQL query in the SQL editor.
5. Click **Run Query** to execute the query.
6. View the results in the results panel.
