# SQL Query Web App - Internship Project

**Description:**
This is my internship project where I built a **web app to write and run SQL queries** using the **Monaco Code Editor**. The app connects to a **PostgreSQL database** and shows query results in real time.

**Features:**

* Write and execute SQL queries
* Connects with PostgreSQL database
* Clean styling using **SCSS**
* **Responsive design** for mobile and desktop

**Technologies Used:**
React | SCSS | Node.js | Express | PostgreSQL | Monaco Editor

**Learning:**

* Learned **PostgreSQL** and backend integration
* Used **Monaco Editor** for SQL
* Built **mobile-first responsive layouts**
* Improved **full-stack development skills**
